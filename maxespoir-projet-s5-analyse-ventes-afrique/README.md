# Projet S5 - Analyse des Ventes en Afrique

## Aperçu
Ce projet vise à analyser les données de ventes d'une boutique en ligne opérant dans plusieurs villes d'Afrique. À travers ce projet, nous allons explorer les ventes, calculer le chiffre d'affaires, identifier les produits les plus vendus, et visualiser les résultats.

## Objectifs
- Générer des données de ventes simulées pour une boutique en ligne.
- Écrire des fonctions pour analyser ces données, y compris le calcul du chiffre d'affaires total, le comptage des commandes par canal, et la détermination des produits les plus vendus.
- Visualiser les résultats de l'analyse à l'aide de graphiques.

## Structure du Projet
- **`Projet_S5_Analyse_Ventes_Afrique.ipynb`**: Contient le code et l'analyse des données de ventes.
- **`README.md`**: Documentation du projet.
- **`requirements.txt`**: Liste des dépendances nécessaires pour exécuter le notebook.

## Instructions
1. Assurez-vous d'avoir Python installé sur votre machine.
2. Installez les dépendances nécessaires en exécutant la commande suivante :
   ```
   pip install -r requirements.txt
   ```
3. Ouvrez le fichier `Projet_S5_Analyse_Ventes_Afrique.ipynb` dans Jupyter Notebook.
4. Exécutez les cellules pour générer les données et effectuer l'analyse.
